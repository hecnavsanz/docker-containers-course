Welcome to nginx!

  • Build Image

    docker build --tag nginx:1.0 --file .\Dockerfile.nginx .

  • Run Container

    docker run -it -d -v .\nginx.default:/etc/nginx/conf.d/default.conf -v .\index.html:/usr/share/nginx/html/index.html --name nginx-cntr -p 8888:80 nginx:1.0

  • Stop Container

    docker stop -s SIGKILL nginx-cntr

  • Remove Container

    docker rm nginx-cntr
